タブバーをページ下部に表示するためのコンポーネントです。ons-tabbar-itemと組み合わせて使うことで、ページを管理できます。

属性

hide-tabs タブを非表示にする場合に指定します。trueもしくはfalseを指定できます。
var JavaScriptからコンポーネントにアクセスするための変数名を指定します。

メソッド

on(eventName,listener) イベントリスナーを追加します。prechangeおよびpostchangeイベントが定義されています。
